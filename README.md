# Protocol 17

Готовый статический сайт для GitHub Pages.

## Как залить

1. Создай новый репозиторий на GitHub.
2. Загрузи в корень репозитория файлы `index.html`, `styles.css`, `script.js`, папку `assets` и файл `.nojekyll`.
3. В GitHub открой `Settings -> Pages`.
4. В `Build and deployment` выбери `Deploy from a branch`.
5. Для branch выбери `main`, для folder выбери `/ (root)`.
6. Сохрани настройки и дождись ссылки от GitHub Pages.
